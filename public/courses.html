Machine Learning

Machine learning which statisticians are calling it statistical learning, we are not entering their battle. Machine learning is tries to find pattern in the data and use that as a learning mechanism to solve later similar task, it does this without being directly programmed to do so,in tradational computer science, in order to do something, we tell computer exactly what to do, but here the case is otherwise,How it learns? we define a learning algorithm which based on that it can learn from the mistakes it makes during learning process, with every mismatch between our answers and what the machine predicts there will be a punishment, based on that punishment the machine learns that it made a big mistake, and next time it tries to improve by making adjustment to they attributes or parameters which had effect on the previous prediction.
